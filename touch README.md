# 🔐 FastAPI Secure File Manager

A secure backend system built using FastAPI.

## 🚀 Features
- User Authentication (2FA OTP)
- File Upload & Download
- File Sharing Between Users
- Security Event Logging
- Basic Threat Detection

## 🛠 Tech Stack
- FastAPI
- SQLite
- Python

## ▶️ Run Project

```bash
uvicorn main:app --reload
```

Open:
http://127.0.0.1:8000/docs

## ⚠️ Note
This is a demo project. Passwords are not hashed.
